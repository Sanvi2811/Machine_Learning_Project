# Machine Learning Model Comparison (From Scratch)

## 📌 Overview

This project focuses on implementing and comparing three fundamental machine learning models from scratch:

* Logistic Regression
* Decision Tree
* Random Forest

The models are evaluated on an imbalanced dataset to understand their performance and behavior.

---

## 📊 Dataset

The models are trained and tested on the **[Your Dataset Name] dataset**.

---

## ⚙️ Models Implemented

* **Logistic Regression**

  * Implemented using Batch Gradient Descent
  * Includes threshold tuning for better performance

* **Decision Tree**

  * Built using Gini Index for splitting
  * Depth control used to reduce overfitting

* **Random Forest**

  * Ensemble of multiple decision trees
  * Uses bootstrap sampling and feature randomness

---

## 📈 Evaluation Metrics

The models are compared using:

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix

---

## 🔍 Results

* Logistic Regression provided a good baseline with balanced performance.
* Decision Tree improved results by capturing non-linear patterns.
* Random Forest achieved the best performance with the highest F1 score and recall.

## 📉 Visualization
